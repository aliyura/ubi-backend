# Integrations and Providers

This folder documents external integrations currently wired in the codebase.

## Structure
- `orchestration.md`: how `ApiProviderService` routes capabilities across providers.
- `providers/*.md`: one file per provider service with capabilities and patterns.
- `platform/*.md`: non-provider integrations (email, file upload/cloud storage).

## Providers Covered
- Bell MFB
- SafeHaven
- VFD Bank
- Flutterwave
- Reloadly
- Dojah
- QoreID
- Smile ID
- Graph
- Termii
- Sendar
- AWS Pinpoint SMS

## Platform Integrations Covered

- Cloudinary + Multer storage
